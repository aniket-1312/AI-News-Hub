# 📰 AI News Hub - Flutter Application

AI News Hub is a clean-architecture Flutter mobile application that fetches and displays Artificial Intelligence related news from a public REST API.  
The app demonstrates real-world API integration, state management, clean project structure, and modern UI practices.

## 📌 Abstract

Artificial Intelligence is rapidly evolving, and staying updated with reliable AI news is essential for students and professionals.  
AI News Hub provides a centralized platform to browse, search, and bookmark AI-related news articles using a real REST API.

---

## ❓ Problem Statement

Students and developers often struggle to find trustworthy and categorized AI news in one place.  
This application solves the problem by aggregating AI news from a public API and presenting it in a user-friendly mobile interface.

---

## 🎯 Objectives

- Integrate a real REST API in Flutter  
- Parse JSON into strongly typed Dart models  
- Display data using ListView and GridView  
- Implement search and favorites  
- Apply clean architecture  
- Build a scalable and maintainable codebase
